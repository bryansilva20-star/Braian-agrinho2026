🌱 Agro Forte, Futuro Sustentável

Projeto web interativo voltado à visualização ambiental e agrícola, com simulação de efeitos climáticos, foco em sustentabilidade e boas práticas de desenvolvimento front-end.

📌 Visão Geral

O Agro Forte, Futuro Sustentável é uma aplicação web desenvolvida como Landing Page interativa, cujo objetivo é demonstrar como fatores climáticos e ambientais influenciam a produção agrícola.

O projeto utiliza um mapa interativo com pontos dinâmicos, simulando condições como temperatura, umidade, estado do solo e variações climáticas. Além disso, promove a conscientização sobre práticas sustentáveis no agronegócio 🌍🌾.

A estrutura inicial foi gerada com apoio de Inteligência Artificial Generativa, sendo posteriormente adaptada, expandida e personalizada manualmente pelo aluno.

🎯 Objetivos do Projeto
Desenvolver habilidades em programação web.
Aplicar conceitos de sustentabilidade ambiental.
Criar visualizações interativas com JavaScript e p5.js.
Trabalhar experiência do usuário (UX) e acessibilidade.
Exercitar criatividade, organização e análise crítica de código.
Utilizar a IA como ferramenta de apoio, de forma consciente e ética.
🛠️ Tecnologias Utilizadas
HTML5 – Estrutura semântica.
CSS3 – Estilização, responsividade e tema escuro.
JavaScript (ES6+) – Interatividade e lógica.
p5.js – Simulação gráfica e mapa interativo.
Inteligência Artificial Generativa – Apoio inicial no desenvolvimento.
